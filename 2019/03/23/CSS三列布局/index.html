CSS三列布局实现

这里的三列布局指的是两边定宽中间自适应的布局，也是面试中最常考的布局方式

圣杯布局

HTML:

CSS:

主要思想是:先将父元素空出左右列的距离，设置三个元素为左浮动，这样就依次排列中、左、右三个元素。由于中间元素的width为100%，所以左右两个元素会被挤到下一行。设置左元素margin-left: -100%这样左元素就可以回退到第一行左端，同理右元素设置margin-left: -300px可以回退到第一行右端。之后分别设置左右元素left: -200px和right: -300px这样就可以将左右元素移动到父元素空出的位置上。
